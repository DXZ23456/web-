## 项目简介 / Project Introduction

### 中文 / Chinese
这是一个功能强大的网页压力测试工具，专门设计用于测试网站和API在高并发访问下的性能表现。支持模拟10-200个并发用户，可以模拟真实用户的访问行为，包括不同的设备类型、浏览器和访问延迟。

**主要功能：**
- 支持10-200个并发用户模拟
- 可配置的测试参数（测试时长、延迟时间、批次大小等）
- 多种设备类型和浏览器UserAgent模拟
- 实时测试进度和结果统计
- 详细的请求日志和性能指标
- 支持GET和POST请求
- 专为短链接生成器等API服务优化

**应用场景：**
- 网站性能测试和容量规划
- API服务压力测试
- 短链接服务性能评估
- 服务器负载能力测试
- 开发和测试环境的性能验证

---

### English / 英语
This is a powerful web stress testing tool specifically designed to test the performance of websites and APIs under high concurrency access. It supports simulating 10-200 concurrent users and can simulate real user access behaviors, including different device types, browsers, and access delays.

**Main Features:**
- Supports 10-200 concurrent user simulation
- Configurable test parameters (test duration, delay time, batch size, etc.)
- Multiple device types and browser UserAgent simulation
- Real-time test progress and result statistics
- Detailed request logs and performance metrics
- Support for GET and POST requests
- Optimized for API services like URL shorteners

**Application Scenarios:**
- Website performance testing and capacity planning
- API service stress testing
- URL shortener service performance evaluation
- Server load capacity testing
- Performance verification in development and test environments
